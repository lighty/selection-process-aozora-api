以下の仕様を満たすサーバとなります

```
・概要
このREST APIサーバーは、青空文庫（https://www.aozora.gr.jp）に存在する作家や作品の情報を提供します。
iOS、Androidアプリや、Webフロントエンドから利用されることを想定しています。

・仕様
作家情報の検索
キーワードから、該当する作家を検索して、作家の以下の情報を返す
作家名
作家名読み
ローマ字表記
生年月日
没年月日
人物概要
WikipediaのURL
作品IDリスト

・作品情報の検索
キーワードから、該当する作品を検索して、作品の以下の情報を返す
作品名
作品名読み
作家名
ファイルのURL
WikipediaのURL
```
